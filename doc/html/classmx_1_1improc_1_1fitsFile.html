template<typename dataT>
class mx::improc::fitsFile< dataT >

Class to manage interactions with a FITS file.

This class wraps the functionality of cfitsio.

Template Parameters
dataTis the datatype to use for in-memory storage of the image. This does not have to match the data type stored on disk.
