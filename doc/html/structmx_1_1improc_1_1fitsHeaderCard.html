mx::improc::fitsHeaderCard Struct Reference
Image Processing » FITS Files

Simple structure to hold the three components of a FITS header card.

Is a struct so all members are public for easy access. Several string-to-number conversions are provided.

template<typename typeT >
typeT mx::improc::fitsHeaderCard::Value ( ) const

Get value converted to the specified type.

Uses convertFromString. typeT can be any fundamental type, or any type which can be typecast from std::string. This is specialized for typeT=std::string so that leading and trailing apostrophes are removed.
