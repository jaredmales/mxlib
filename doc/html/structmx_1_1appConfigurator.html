template<typename typeT >
typeT mx::appConfigurator::get ( const std::string &  name)

Get the final value of the target, returned as the specified type.

The default-constructed/initialized value is returned if isSet() == false, so this should carefully be used in conjunction with isSet() to avoid overwriting default values.

This works for vectors as well (through tag dispatching). In any case, you must explicitly specify typeT, as in

float f = get<float>(name);
std::vector<float> v = get<std::vector<float>>(name);
Return values
0on success.
-1on error.
Parameters
[in]namethe config target name
