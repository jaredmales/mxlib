Fill in an Eigen-like array with a circular pupil mask.

Sets any pixel which is at rad <= r < rad+(1.0/overscan) pixels to rho = 1,

Parameters
[out]mis the allocated Array. Dimensions are used to create the pupil.
[in]eps[optional] is the central obscuration. 0-1, default is 0.
[in]rad[optional] is the desired radius. If rad <= 0, then the maximum radius based on dimensions of m is used.
Return values
0on success
-1on error
