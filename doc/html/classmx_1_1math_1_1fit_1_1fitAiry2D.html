template<typename fitterT>
class mx::math::fit::fitAiry2D< fitterT >

Class to manage fitting a 2D Airy pattern to data via the levmarInterface.

In addition to the requirements on fitterT specified by levmarInterface this class also requires this definition in fitterT

static const int nparams = 5;

where the number 5 is replaced by the number of parameters that fitterT expects to fit.

fitterTa type meeting the above requirements.
